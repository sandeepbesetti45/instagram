# 📸 Instagram Clone (Authentication + Dashboard)

A mini Instagram clone built using **React**, **Firebase Authentication**, and **Redux Toolkit**. This project focuses on implementing a complete user authentication flow and displaying a dashboard UI after login — similar to Instagram's homepage experience.

---

## 🚀 Features

- 🔐 Firebase Authentication for SignUp & Login
- ⚙️ Redux Toolkit for global state management
- ✅ Session persistence with `onAuthStateChanged()`
- 🧠 Conditional rendering based on login status
- 🧩 Reusable components: Navbar, Sidebar, Posts, Suggestions
- 👤 Avatar displaying first letter of username
- 🌀 Loading spinner during authentication check

---

## 🧠 What I Learned

- How to implement secure login/signup using Firebase
- Managing global state using Redux Toolkit (`dispatch`, `useSelector`)
- Protecting routes and persisting login state across refreshes
- Structuring React projects into clean, modular components
- Dynamically rendering UI based on Redux state and Firebase data
